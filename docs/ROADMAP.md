# NutriScan Roadmap

## North Star

Ship a consumer-grade nutrition app where the user can scan food, understand nutritional impact immediately, and build a persistent health journey over time.

## 30 Days

- production-ready frontend structure
- Supabase auth live
- onboarding live
- scan flow live
- history live
- deployable environment configuration

## 60 Days

- dashboard with daily progress
- correction flow for inaccurate scans
- retention analytics
- monitoring and alerting
- basic automated test coverage

## 90 Days

- weekly and monthly insights
- recommendation engine layer
- monetizable premium surface
- stronger growth instrumentation

## Engineering KPIs

- scan success rate
- scan time to result
- onboarding completion rate
- day-1 and day-7 retention
- failed request rate
- percentage of scans saved successfully
