# Decisoes Tecnicas

> Registro de decisoes arquiteturais e tecnicas do projeto.
> O mais importante aqui e o **por que**, nao o **o que**.
> Use o template `Templates/Template-Decisao.md` para novas decisoes.

---

## Indice

### Arquitetura
- CSS custom design system (sem Tailwind) - decisao da Fase 1
- Supabase como backend (PostgreSQL + RLS + Auth)
- Serverless functions na Vercel (api/)

### IA
- Fallback em cascata: Groq → Anthropic → Ollama
- Analise de scan via visao (Claude/llava)

### Design
- Mobile-first (max-width 480px)
- Temas: Polar (branco editorial) e iOS Premium (true black)
- SVG icons no lugar de emojis (commit f82fa61)
