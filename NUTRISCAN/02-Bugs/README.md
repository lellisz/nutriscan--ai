# Base de Bugs Resolvidos

> Cada bug vira uma nota aqui. Quando aparecer algo parecido no futuro, busque aqui primeiro.
> Use tags para categorizar: `#bug`, `#css`, `#api`, `#auth`, `#supabase`, `#router`, etc.
> Use o template `Templates/Template-Bug.md` para novos bugs.

---

## Como usar

1. Resolveu um bug? Crie uma nota aqui usando o template
2. Bug parecido apareceu? Busque por tags ou palavras-chave
3. Iniciando sessao com IA? Mencione bugs relevantes no prompt
