# Caderno de Estudo — Praxis Nutri

> Aprendizados extraidos de cada sessao de desenvolvimento. Termos, conceitos, logicas de programacao, padroes e licoes de bugs reais.

## Como funciona
- Ao final de cada sessao, a skill `/study` gera notas com os conceitos abordados
- Cada nota explica o "por que" alem do "como"
- Conectado ao vault Obsidian para criar uma base de conhecimento viva
